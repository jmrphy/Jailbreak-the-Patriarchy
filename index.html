Jailbreak the Mass Media

Google Chrome extension to expose mass media concentration by replacing subsidiary media companies with the name of their parent company

Jailbreak the Mass Media is a fork of "Jailbreak the Patriarchy" by Danielle Sucher. Suggestions and pull requests welcome.

Known issues:

  • The subsidiary/parent data and mappings are all hard-coded but it would be nice to keep them up to date dynamically. Note there is a directory "scraper," which contains a few aborted efforts to write a scraper that could pull this data from Wikipedia pages. If you have any clue about how one could do something like that, please do let me know.
